# Water Monitoring System

IoT-based water quality monitoring platform for aquaculture and hydroponics.

## Architecture
- ESP32 sensor nodes
- MQTT communication
- Raspberry Pi 5 gateway
- Dashboard analytics
- Alert engine

## Monitored Parameters
- pH
- Dissolved Oxygen (DO)
- Temperature
- Conductivity
- TDS
- ORP
- Ammonia
- Nitrate
- Nitrite
